# Interactive Phishing Awareness Training Platform 🛡️

## Project Overview
An interactive cybersecurity awareness platform designed to teach users how to identify and defend against phishing attacks and social engineering threats. This project provides a complete, hands-on learning experience with interactive modules, real-world case studies, and dynamic quizzes.

## 🚀 Features
- **Modern Cyber UI:** Dark mode with neon blue and green accents, glassmorphism cards, and smooth animations.
- **8 Interactive Modules:** Covers everything from basic phishing concepts to spotting fake URLs and understanding social engineering tactics.
- **Interactive "Spot the Phish":** Clickable email templates where users must find hidden red flags.
- **Dynamic Progress Tracking:** Uses `localStorage` to save user progress and quiz scores locally (no database required).
- **Final Assessment Quiz:** 15 randomized multiple-choice questions with instant feedback and explanations.
- **Certificate Generation:** Automatically generates a personalized, printable completion certificate for users who score 80% or higher.

## 🛠️ Technology Stack
- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom animations, responsive design, and modern UI variables (No external frameworks like Bootstrap or Tailwind).
- **Vanilla JavaScript:** Logic for quizzes, progress tracking, DOM manipulation, and certificate generation (No React or jQuery).

## 📂 Folder Structure
The codebase is clean and modular:
- `index.html` - Landing page and progress dashboard.
- `modules.html` - The core interactive learning content.
- `quiz.html` - The randomized assessment.
- `certificate.html` - Dynamic certificate generator.
- `css/` - Contains `style.css`, `animations.css`, and `responsive.css`.
- `js/` - Contains logic files (`main.js`, `quiz.js`, `progress.js`, `certificate.js`).

## 💻 How to Run Locally
1. Clone or download this repository.
2. Extract the folder.
3. Simply double-click `index.html` to open it in any modern web browser. 
4. No server, Node.js, or backend setup is required!
