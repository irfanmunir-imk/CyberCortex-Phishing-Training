# Interactive Phishing Awareness Training Platform

## Project Overview
A complete, interactive cybersecurity awareness platform designed to teach users how to identify and defend against phishing attacks and social engineering.

## Features
- **Modern Cyber UI:** Dark mode, glassmorphism, neon glows (blue/green).
- **Interactive Modules:** 8 comprehensive learning modules with clickable examples, hover reveals, and real-world case studies.
- **Progress Tracking:** LocalStorage-based tracking for module completion and quiz scores.
- **Dynamic Quiz:** 15 randomized multiple-choice questions with instant feedback.
- **Certificate Generation:** Automatically generated, printable completion certificate.
